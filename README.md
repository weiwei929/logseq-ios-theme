# Logseq iOS 风格主题

一个模仿 iOS 设计风格的 Logseq 主题，提供明亮和暗黑两种模式。

## 特性

- 🎨 iOS 风格界面设计
- 🌓 完美适配明暗两种模式
- 📱 响应式布局，支持移动端
- ✨ 平滑过渡动画效果
- 🎯 优化的交互体验
- 🖥 清晰的文本显示

## 安装

1. 下载 `custom.css` 文件
2. 将文件放入 Logseq 配置目录下
3. 重启 Logseq 即可生效

## 自定义

主题使用 CSS 变量，你可以通过修改 `:root` 中的变量来自定义主题样式：

```css
:root {
  --bg-color: #f0f0f0;              /* 背景色 */
  --primary-color: #8b919a;         /* 主色调 */
  --accent-color: #ff2d55;          /* 强调色 */
  /* 更多变量... */
}
